# LangChain Experiments

## OpenAI API Key
* https://platform.openai.com/account/api-keys

## Python
*Python 3.11+*

Create a .env file with:
```
OPENAI_API_KEY=YOUR_OPENAI_API_KEY_GOES_HERE
```

Run the following in Terminal:
```
cd python
pip install -r requirements.txt
python3 index.py
```

## JavaScript
*Node.js 18+*

Create a .env file with:
```
OPENAI_API_KEY=YOUR_OPENAI_API_KEY_GOES_HERE
```

Run the following in Terminal:
```
node index.js
```


## TypeScript
*Node.js 18+*

Create a .env file with:
```
OPENAI_API_KEY=YOUR_OPENAI_API_KEY_GOES_HERE
```

Run the following in Terminal:
